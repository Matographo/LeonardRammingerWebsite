# Axio

Axio ist eine ausdrucksstarke, integrierbare und optimierte Allzwecksprache. Durch die Verwendung einer speziellen Syntax für bestimmte Bereiche ist sie für Programmierer leicht verständlich und gleichzeitig hoch optimiert. Zu diesen Bereichen zählen das Erstellen von Automaten, eine sprechendere Syntax für mathematische Ausdrücke und eine große Palette an Metaprogramming-Befehlen.

# Entwicklungsstand

Momentan ist die Sprache in der Entwicklung.

Falls du ein Teil von Axio sein willst, kannst du dich mit der Idee der Sprache vertraut machen und teile des Compilers mit compilieren.
[Axio Repo](https://github.com/Coditary/Axio)

Um die Sprache universell zu machen, ist die Sprache in dem Repo Englisch.

Ich gebe mein bestes, dass Axio in Github so organisiert ist, dass es Tasks gibt, welche erfüllt werden können, so dass auch neueinsteiger wissen wo sie was ändern können.
